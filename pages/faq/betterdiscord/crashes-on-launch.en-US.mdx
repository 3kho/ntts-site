# After installing BetterDiscord, Discord crashes on launch

<Callout emoji="ℹ️">
    This answer will work for Windows users only.
</Callout>

<Steps>

#### Go to BetterDiscord folder

Press `Win + R` and type `%APPDATA%/betterdiscord`.

#### Look for the following files and delete them

```
0PluginLibrary.plugin
0PluginLibrary.plugin.json
0PluginLibrary.config.json
0PluginLibrary.data.json
0PluginLibrary.raw
ZeresPluginLibrary.plugin
ZeresPluginLibrary.json
ZeresPluginLibrary.config.json
ZeresPluginLibrary.raw
ZeresPluginLibrary.data.json
```

<Callout emoji="ℹ️">
    You might not have all these files, so just delete the ones you have.
</Callout>


#### Restart Discord

Restart Discord via the [system tray](https://www.tutorialspoint.com/windows10/windows10_system_tray.htm) icon.

#### Download plugin

Download [this plugin](https://raw.githubusercontent.com/rauenzi/BDPluginLibrary/master/release/0PluginLibrary.plugin.js). Open the hyperlink, and Press `Ctrl + S` to download it.

#### Save file

Copy and paste the downloaded file to your plugins folder, then restart Discord
again via the [system tray](https://www.tutorialspoint.com/windows10/windows10_system_tray.htm) icon.

</Steps>
That's all. Discord should no longer crash on launch!
